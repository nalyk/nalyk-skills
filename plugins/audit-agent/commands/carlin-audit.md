## THE 13 QUESTIONS

1. **HIDDEN AGENDA DETECTOR** — "What's the REAL reason this exists?" Not the marketing reason. Whose problem does it ACTUALLY solve — users' or creators'?
2. **EUPHEMISM EXPOSURE** — "What language are we using to avoid admitting uncomfortable truths?" "Freemium" = we'll annoy you until you pay. What comfortable lies are baked into the descriptions?
3. **FOLLOW THE MONEY** — "Who ACTUALLY benefits — users, or the people building it?" Does this exist because users need it, or because it looks good in a demo?
4. **THE QUIET PART** — "What truth would we NEVER put on the landing page?" The real limitations, the actual use case, the customers you don't want.
5. **CONTRADICTION DETECTOR** — "What contradictions exist that everyone pretends don't?" "We value privacy" + tracking everything. "Simple and powerful" + 47 settings. Find the hypocrisy.
6. **ABSURDITY CHECK** — "How is this fundamentally absurd when you step back?" What would make an alien anthropologist laugh? Where is everyone playing pretend?
7. **PERFORMANCE VS REALITY** — "What are we PERFORMING versus actually delivering?" "AI-powered" = one API call. "Platform" = a WordPress site. Where's the theater?
8. **OUTSIDER PERSPECTIVE** — "What would someone with ZERO context think?" Not your target user — your grandmother. What would genuinely confuse or disturb them?
9. **MEANINGLESS RULES** — "What 'best practices' make zero sense for THIS situation?" Blindly copied competitor moves, "that's how it's done" without why. Find the cargo cult.
10. **EMBARRASSMENT TEST** — "What truth would embarrass us if users knew?" The shortcut, the debt, the feature nobody uses, the metric that's actually terrible.
11. **HYPOCRISY AUDIT** — "How does this contradict our stated values?" "We hate spam" but 5 emails a week. "Transparent pricing" with hidden fees. Expose it.
12. **HERD MENTALITY CHECK** — "What are we doing ONLY because everyone else does?" The feature nobody asked for but competitors have.
13. **THE ULTIMATE REDUCTION** — "If we removed ALL the bullshit, what would actually be left?" Strip buzzwords, unused features, vanity metrics. Is there anything valuable underneath?
