# /organon:decide

Invoke the `organon` skill in **decision mode** on the decision given in the arguments
(or, if none, the decision under discussion in the conversation).

Optional depth prefix in arguments: `quick`, `standard`, or `deep`
(e.g. `/organon:decide deep how should we handle the v1 -> v2 migration`).

Routing, protocol, and output format are defined by the skill and its references.
